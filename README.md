## 🎙️ ক্রস-ল্যাঙ্গুয়েজ ভয়েস টাইপিং (কিভাবে কাজ করে?)
মোবাইল দিয়ে পিসিতে রিয়েল-টাইম ভয়েস টাইপ করার এক যুগান্তকারী ফিচার!
1. কীবোর্ড থেকে **ওয়াইফাই ভয়েস টাইপিং** অপশনে ক্লিক করুন।
2. স্ক্রিনে আসা **QR কোডটি** আপনার মোবাইল দিয়ে স্ক্যান করুন (পিসি ও মোবাইল একই ওয়াইফাই বা হটস্পটে থাকতে হবে)।
3. মোবাইলের ব্রাউজারে একটি পেজ ওপেন হবে। সেখানে থাকা টেক্সটবক্সে ক্লিক করে আপনার **মোবাইলের কীবোর্ডের মাইক (🎤) আইকনে** চাপ দিন।
4. এবার আপনি **বাংলা, ইংরেজি, আরবি, হিন্দি, স্প্যানিশ—যেকোনো ভাষায়** কথা বলুন। মোবাইলের কীবোর্ড যা ইনপুট দিবে, সেটি চোখের পলকে আপনার পিসিতে টাইপ হয়ে যাবে!
