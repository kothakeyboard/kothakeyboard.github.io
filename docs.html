📖 Bangla Keyboard CS - User Manual

স্বাগতম! Bangla Keyboard CS উইন্ডোজ (Windows 10/11) এর জন্য তৈরি করা এযাবৎকালের সবচেয়ে আধুনিক, ফাস্ট এবং ফিচার-প্যাকড বাংলা কীবোর্ড। ৯ লক্ষ+ শব্দের বিশাল ডিকশনারি, স্মার্ট অটো-সাজেশন, আরবি লেআউট, ক্রস-ল্যাঙ্গুয়েজ ওয়াইফাই ভয়েস টাইপিং এবং জিরো-অ্যালোকেশন মেমোরি ম্যানেজমেন্ট— সবকিছু মিলিয়ে এটি আপনার টাইপিংকে করবে সুপারফাস্ট!

🌟 অন্যান্য কীবোর্ড থেকে এটি কেন সেরা?

  • কার্সরের পজিশন ফিক্স: ক্রোম, এজ, ভিএস কোড, মেসেঞ্জারে সাজেশন বার এখন থেকে ঠিক আপনার কার্সরের নিচেই শো করবে!
  • ডিকশনারি কাস্টমাইজেশন: ভুল কোনো শব্দ সাজেশনে আসলে তার ওপর রাইট-ক্লিক (Right-Click) করেই ডিকশনারি থেকে তা মুছে ফেলা যায়।
  • সুপারফাস্ট পারফরম্যান্স: C++ এর নেটিভ Range-loop ব্যবহারের কারণে এটি র‍্যাম একদমই টানে না।

📥 ২. ডাউনলোড এবং ইনস্টলেশন

  1. এখান থেকে সফটওয়্যারটি ডাউনলোড করুন।
  2. ডাউনলোড করা .exe ফাইলটিতে ডাবল ক্লিক করুন। (উইন্ডোজ ডিফেন্ডার ওয়ার্নিং দিলে More Info > Run Anyway তে ক্লিক করুন)।
  3. ইনস্টল হয়ে গেলে টাস্কবারের নিচে ডানদিকে (System Tray) কীবোর্ডের আইকন দেখতে পাবেন। সেখানে ক্লিক করলেই মূল উইন্ডো চলে আসবে।

⌨️ ৩. কীবোর্ড শর্টকাট

  • Ctrl + 1 : বাংলা ইউনিকোড
  • Ctrl + 2 : ইংরেজি
  • Ctrl + 3 : বিজয় আন্সি
  • Ctrl + 4 : বাংলা ফনেটিক
  • Ctrl + 5 : আরবি
  • Ctrl + Space : বাংলা ও ইংরেজির মাঝে সুইচ (Toggle) করতে।

🗺️ ৪. লেআউট ম্যাপিং (Interactive)

নিচের বাটনগুলোতে ক্লিক করে কীবোর্ডের শিফট (Shift) ক্যারেক্টারগুলো দেখে নিতে পারেন।

🎙️ ৫. ওয়াইফাই ভয়েস টাইপিং

পিসিতে মাইক্রোফোন ছাড়াই মোবাইল দিয়ে স্ক্যান করে বাংলা, ইংরেজি ও আরবিসহ যেকোনো ভাষায় কথা বললে পিসিতে টাইপ হবে!

  1. কীবোর্ডের ট্রে মেনু থেকে ওয়াইফাই ভয়েস টাইপিং চালু করুন।
  2. স্ক্রিনে আসা QR কোডটি মোবাইল দিয়ে স্ক্যান করুন।
  3. মোবাইলে ওপেন হওয়া পেজে মাইক (🎤) আইকনে চাপ দিয়ে কথা বলুন।

🧰 ৬. অন্যান্য টুলস

  • লোকাল ফাইল ট্রান্সফার: কোনো ক্যাবল বা ডেটা খরচ ছাড়াই পিসি থেকে মোবাইলে ফাইল আদান প্রদান করুন (ওয়াইফাই রাউটারের স্পিডে)।
  • লাইভ কনভার্টার: ইউনিকোড ↔ বিজয় আন্সি (SutonnyMJ) লাইভ কনভার্ট করুন।
  • কাস্টম লেআউট: সেটিংস থেকে নিজের ইচ্ছেমতো ৩টি কীবোর্ড লেআউট তৈরি করুন।
