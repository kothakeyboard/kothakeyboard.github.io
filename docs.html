📖 Kotha Keyboard - User Manual

উইন্ডোজের জন্য তৈরি করা এযাবৎকালের সবচেয়ে আধুনিক, ফাস্ট এবং ফিচার-প্যাকড বাংলা কীবোর্ড। ৯ লক্ষ+ শব্দের বিশাল ডিকশনারি, স্মার্ট অটো-সাজেশন, ক্রস-ল্যাঙ্গুয়েজ ওয়াইফাই ভয়েস টাইপিং এবং জিরো-অ্যালোকেশন মেমোরি ম্যানেজমেন্ট— সবকিছু মিলিয়ে এটি আপনার টাইপিংকে করবে সুপারফাস্ট!

📥 ২. ডাউনলোড এবং ইনস্টলেশন

  1. এখান থেকে সফটওয়্যারটি ডাউনলোড করুন।
  2. ডাউনলোড করা .exe ফাইলটিতে ডাবল ক্লিক করুন। (উইন্ডোজ ডিফেন্ডার ওয়ার্নিং দিলে More Info > Run Anyway)।
  3. ইনস্টল হয়ে গেলে ডেক্সটপে কীবোর্ড এর টপবার দেখতে পাবেন, মাইনাস আইকনে ক্লিক করলে নিচে টাস্কবারের ডানদিকে (System Tray) কীবোর্ডের আইকন দেখতে পাবেন।

⌨️ ৩. কীবোর্ড শর্টকাট ও সেটিংস

  • Ctrl + 1 : বাংলা ইউনিকোড
  • Ctrl + 2 : ইংরেজি
  • Ctrl + 3 : বিজয় আন্সি
  • Ctrl + 4 : বাংলা ফনেটিক
  • Ctrl + 5 : আরবি
  • Ctrl + Space : বাংলা ও ইংরেজির মাঝে টগল করতে।
  • Ctrl + 0 : ফনেটিক মোড টগল (Unicode <-> ANSI) করতে।

⚙️ সেটিংস উইন্ডোর টগল অপশনসমূহ:

  • ডিফল্ট শর্টকাট ব্যবহার করুন: এটি চালু থাকলে উপরের ডিফল্ট শর্টকাটগুলো কাজ করবে। এটি বন্ধ করে আপনি নিজের ইচ্ছেমতো (যেমন Shift+A, Alt+1 ইত্যাদি) শর্টকাট সেট করতে পারবেন।
  • ইংলিশ লেআউটেও সাজেশন দেখান: এটি চালু থাকলে ইংরেজি টাইপ করার সময়ও ডিকশনারি থেকে অটো-সাজেশন আসবে।
  • সাজেশনে টাইপ করা ইংলিশ টেক্সট দেখান: টাইপিংয়ের সময় আপনি ইংরেজিতে যা টাইপ করছেন (যেমন: 'Ami') সেটিও সাজেশনের শেষে প্রদর্শিত হবে, যাতে প্রয়োজন হলে সরাসরি ইংরেজি শব্দটিও বসানো যায়।
  • প্রতিটি উইন্ডোর জন্য আলাদা লেআউট মনে রাখুন: এটি চালু করলে, আপনি Chrome এ বাংলা টাইপ করে Microsoft Word এ গেলে সেখানে আগে যে লেআউট ছিল (যেমন ইংরেজি) সেটি অটোমেটিক সিলেক্ট হয়ে যাবে।

💡 ৪. সাজেশন বার গাইড ও টিপস

Kotha Keyboard এর সবচেয়ে পাওয়ারফুল ফিচার হলো এর অটো-সাজেশন বার।

  • নেভিগেশন: টাইপ করার সময় সাজেশন আসলে কীবোর্ডের Up/Down Arrow (উপর/নিচ) কী চেপে সিলেক্ট করতে পারবেন এবং Enter, Tab বা Space চেপে শব্দটি বসাতে পারবেন, অথবা সিলেক্ট করে যাস্ট স্পেস দিলেই সেটি অ্যাপ্লাই হয়ে যাবে!
  • মাউস ক্লিক: মাউস দিয়ে ক্লিক করেও শব্দ ইনসার্ট করা যায়।
  • ডিকশনারি থেকে শব্দ মুছুন: অনেক সময় ভুল বা অপ্রয়োজনীয় শব্দ সাজেশনে আসতে পারে। সেই শব্দের উপর মাউসের রাইট-ক্লিক (Right-Click) করে "ডিকশনারি থেকে মুছুন" এ ক্লিক করলে সেটি স্থায়ীভাবে ব্ল্যাকলিস্ট হয়ে যাবে এবং আর কখনো সাজেশনে আসবে না।

⌨️ ৫. বিজয় (Unicode ও ANSI) টাইপিং গাইড

সেটিংস উইন্ডোতে "ট্রাডিশনাল টাইপিং" নামে একটি অপশন রয়েছে, যা আপনার টাইপিং স্টাইল নির্ধারণ করে।

🔴 ট্রাডিশনাল টাইপিং OFF (Smart Mode) - [ডিফল্ট]

স্মার্ট মোডে, আপনি কীবোর্ডে যেভাবে আঁকা আছে বা স্ক্রিনে যেভাবে দেখা যায় সেভাবেই টাইপ করবেন। অর্থাৎ, যেসব কার অক্ষরের বামে বসে (যেমন: ি, ে, ৈ), সেগুলো অক্ষরের আগেই টাইপ করতে হবে।

  • কি: ি + ক (w + D)
  • কে: ে + ক (c + D)
  • যুক্তবর্ণ: ক + ্ + ষ (J + G + N) = ক্ষ

নোট: আপনি আগে কার টাইপ করলেও কীবোর্ড নিজে থেকেই ইন্টারনালি ইউনিকোড ফরম্যাট ঠিক করে নেবে!

🟢 ট্রাডিশনাল টাইপিং ON (Classic/Pronunciation Mode)

এই মোডে আপনি উচ্চারণ অনুযায়ী টাইপ করবেন। কার সবসময় অক্ষরের পরেই বসবে, কিন্তু কীবোর্ড নিজে থেকেই কারটিকে অক্ষরের আগে নিয়ে যাবে!

  • কি: ক + ি (D + w)
  • কে: ক + ে (D + c)
  • কো: ক + ে + া (D + c + v)

নোট: ট্রাডিশনাল টাইপিং অফ বা অন যাই থাকুক না কেন, আমাদের স্মার্ট ডিকশনারি সঠিকভাবে শব্দ রিড করতে পারবে এবং সাজেশন দেখাবে!

🗣️ ৬. ফনেটিক টাইপিং গাইড ও মডিফায়ার

ফনেটিক টাইপিংয়ের ক্ষেত্রে যুক্তবর্ণ, কার এবং স্পেশাল ক্যারেক্টার লেখার কিছু স্মার্ট নিয়ম রয়েছে। নিচে সবগুলো শর্টকাট এবং ট্রিকস তুলে ধরা হলো:

১. যুক্তবর্ণ তৈরি ও ভাঙা (Zero Width Non-Joiner)

  • যুক্তবর্ণ তৈরি: পাশাপাশি দুটি ব্যঞ্জনবর্ণ লিখলে তা স্বয়ংক্রিয়ভাবে যুক্ত হয়ে যায়। যেমন: shikkha = শিক্ষা।
  • যুক্তবর্ণ ভাঙা: যদি দুটি ব্যঞ্জনের মাঝে যুক্তবর্ণ না চান, তবে তাদের মাঝে o (ও) চাপুন। এটি একটি ইনভিজিবল কার হিসেবে কাজ করে। যেমন: kko = ক্ক, কিন্তু koko = কক।
  • ZWNJ বা জয়েনার ভাঙা: আপনি চাইলে // (ডাবল স্ল্যাশ) অথবা ~ (Tilde) কী চেপেও যুক্তবর্ণ ভাঙতে পারেন। যেমন: kor//te = কর‌তে।
  • হসন্ত + ZWNJ: দুটি ব্যঞ্জনের মাঝে স্পেস না দিয়েও আলাদা দেখাতে চাইলে `` (ডাবল ব্যাকটিক) চাপুন। এটি 'হসন্ত + ZWNJ' বসাবে। যেমন: kos``meTiks = কস্‌মেটিকস্।
  • সাধারণ স্ল্যাশ: ৩ বার স্লাশ /// দিলে সেটি সাধারণ // হিসেবে প্রিন্ট হবে।

২. হসন্ত, রেফ এবং র-ফলা

  • হসন্ত (্): হসন্ত দিতে q অথবা ` (Backtick) অথবা *g ব্যবহার করুন।
  • র-ফলা (্র): ব্যঞ্জনবর্ণের পরে 'r' লিখলে র-ফলা হবে। যেমন: kro = ক্র।
  • রেফ (র্): ব্যঞ্জনবর্ণের আগে 'rr' লিখলে রেফ হবে। যেমন: rrk = র্ক।
  • ঋ-কার (ৃ): 'rri' লিখলে ঋ-কার হবে। যেমন: krri = কৃ।

৩. য-ফলা এবং ব-ফলা

  • য-ফলা (্য): ব্যঞ্জনবর্ণের পর z, y বা Z (Shift+Z) লিখলে য-ফলা হবে। যেমন: kz = ক্য। (Z সবসময় ফোর্সড য-ফলা হিসেবে কাজ করবে)।
  • ব-ফলা (্ব): ব্যঞ্জনবর্ণের পর w লিখলে ব-ফলা হবে। যেমন: sw = স্ব। (কিন্তু শুধু 'w' লিখলে 'ও' হবে)।

৪. স্মার্ট ম্যাক্রো ও স্পেশাল ক্যারেক্টার

  • ক্স / এক্স (x/X): শব্দের শুরুতে 'x' বা 'X' লিখলে হবে এক্স (যেমন: xray = এক্সরে)। কিন্তু শব্দের মাঝে বা শেষে লিখলে হবে ক্স (যেমন: box = বক্স)।
  • জ-ফলা আকার (্যা/অ্যা): এর জন্য ae ব্যবহার করুন। যেমন: gyan = জ্ঞান, byatha = ব্যাথা।
  • অ্যা / এ্যা: aya = অ্যা, eya = এ্যা।
  • র‌্যা: rya অথবা rza লিখলে র‌্যা হবে।
  • ওয়া / ওয়ে / ওয়্যা: wa = ওয়া, we = ওয়ে, wae = ওয়্যা।

৫. যতিচিহ্ন (Punctuation) ও অল্টারনেট শর্টকাট

  • দাঁড়ি এবং ফুলস্টপ: একটি ডট . দিলে বাংলা দাঁড়ি (।) আসবে। আর ইংরেজি ফুলস্টপ লিখতে চাইলে ডাবল ডট .. দিন।
  • বিসর্গ এবং কোলন: একটি কোলন : দিলে বিসর্গ (ঃ) আসবে। আর ইংরেজি কোলন লিখতে চাইলে ডাবল কোলন :: দিন।
  • টাকা (৳): ডলার সাইন $ চাপলে টাকার চিহ্ন (৳) আসবে।
  • খণ্ড-ত (ৎ): TH অথবা *t লিখুন।
  • অনুস্বার (ং): ng ।
  • বিসর্গ (ঃ): : অথবা hh অথবা *l ।
  • চন্দ্রবিন্দু (ঁ): ^ অথবা A (Capital A) অথবা *a ।

🕌 ৭. আরবি (Arabic) টাইপিং গাইড

আমাদের কীবোর্ডে সম্পূর্ণ নিখুঁত একটি আরবি লেআউট রয়েছে, সাথে হারাকাত (যবর, যের, পেশ) সাপোর্ট এবং বিশেষ শর্টকাট।

১. সাধারণ অক্ষর (Basic Letters)

ইংরেজি অক্ষরের ধ্বনির সাথে মিলিয়ে আরবি অক্ষর সাজানো হয়েছে।

a = ا, b = ب, t = ت, Z = ث, j = ج, H = ح, K = خ, d = د, D = ذ, r = ر, z = ز, s = س, S = ش, c = ص, C = ض, p = ط, P = ظ, ' = ع, g = غ, f = ف, q = ق, k = ك, l = ل, m = م, n = ن, h = ه, w = و, y = ي

হামযাহ ও আলিফের বিশেষ রূপ: x = ء, E = أ, W = ؤ, I = إ, O = ئ, Y = ى (আলিফ মাকসুরা), T = ة (গোল তা)।

২. হারাকাত (যবর, যের, পেশ) ও চিহ্ন

অক্ষর টাইপ করার পর হারাকাত দিতে হয়।

  • যবর (َ): e
  • যের (ِ): i
  • পেশ (ُ): u
  • খাড়া যবর (ٰ): A (Shift+a)
  • খাড়া যের (ٖ): I দুইবার চাপলে (ইচ্ছাকৃতভাবে ডাবল I)
  • উল্টা পেশ (ٗ): U (Shift+u)
  • তানভীন: ডাবল টাইপ করলে তানভীন হবে। যেমন ee চাপলে দুই যবর (ً) হবে।
  • তাশদীদ (ّ): v চাপলে তাশদীদ হবে।
  • সুকুন / জযম (ْ): o চাপলে জযম হবে।
  • মাদ (ٓ): ~ (Tilde) চাপলে মাদ হবে।
  • উপরে ও নিচে হামযা: < = ٔ এবং > = ٕ

৩. স্মার্ট ম্যাক্রো (আল্লাহ)
