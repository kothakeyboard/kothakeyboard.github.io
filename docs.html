📖 Bangla Keyboard CS - User Manual

উইন্ডোজের জন্য তৈরি করা এযাবৎকালের সবচেয়ে আধুনিক, ফাস্ট এবং ফিচার-প্যাকড বাংলা কীবোর্ড। ৯ লক্ষ+ শব্দের বিশাল ডিকশনারি, স্মার্ট অটো-সাজেশন, ক্রস-ল্যাঙ্গুয়েজ ওয়াইফাই ভয়েস টাইপিং এবং জিরো-অ্যালোকেশন মেমোরি ম্যানেজমেন্ট— সবকিছু মিলিয়ে এটি আপনার টাইপিংকে করবে সুপারফাস্ট!

🚀 ২. কেন এই কীবোর্ডটি আলাদা? (Features)

অন্যান্য সাধারণ কীবোর্ডের তুলনায় Bangla Keyboard CS সম্পূর্ণ নতুন প্রযুক্তিতে তৈরি, যা আপনার পিসির পারফরম্যান্স ঠিক রেখে সর্বোচ্চ ফিচার প্রদান করে।

⚡ সুপারফাস্ট পারফরম্যান্স

সম্পূর্ণ C++ নেটিভ উইন্ডোজ API দিয়ে তৈরি। এর Zero-Allocation মেমোরি ম্যানেজমেন্টের কারণে টাইপিংয়ের সময় কোনো ল্যাগ বা পিসি হ্যাং হওয়ার সম্ভাবনা নেই।

🧠 স্মার্ট অটো-সাজেশন

৯ লক্ষ ২১ হাজার শব্দের বিশাল অফলাইন ডিকশনারি! আপনি ভুল টাইপ করলেও এটি নেক্সট-ওয়ার্ড প্রেডিকশন এবং ফনেটিক ইন্টেলিজেন্সের মাধ্যমে সঠিক শব্দটি সাজেস্ট করবে।

🎙️ ওয়াইফাই ভয়েস টাইপিং

পিসিতে মাইক্রোফোন নেই? কোনো সমস্যা নেই! মোবাইল দিয়ে QR কোড স্ক্যান করে মোবাইলকেই মাইক্রোফোন বানিয়ে বাংলা, ইংরেজি বা আরবিতে রিয়েল-টাইম টাইপ করুন।

🔤 মাল্টি-ল্যাঙ্গুয়েজ ও আরবি সাপোর্ট

বাংলা ইউনিকোড, ফনেটিক এবং বিজয় আন্সির (SutonnyMJ) পাশাপাশি রয়েছে সম্পূর্ণ নিখুঁত আরবি (Arabic) কীবোর্ড লেআউট এবং হারাকাত (যবর, যের, পেশ) সাপোর্ট।

🛠️ কাস্টম লেআউট ক্রিয়েটর

সেটিংস থেকে আপনার নিজের পছন্দমতো শর্টকাট দিয়ে সর্বোচ্চ ৩টি পর্যন্ত সম্পূর্ণ কাস্টম কীবোর্ড লেআউট তৈরি করার সুবিধা।

🔄 লাইভ কনভার্টার ও ফাইল ট্রান্সফার

সরাসরি ইউনিকোড ↔ বিজয় আন্সি লাইভ কনভার্টার এবং ক্যাবল ছাড়াই মোবাইল-টু-পিসি লোকাল নেটওয়ার্কে ফাইল ট্রান্সফার করার বিল্ট-ইন টুল।

📥 ৩. ডাউনলোড এবং ইনস্টলেশন

  1. এখান থেকে সফটওয়্যারটি ডাউনলোড করুন।
  2. ডাউনলোড করা .exe ফাইলটিতে ডাবল ক্লিক করুন। (উইন্ডোজ ডিফেন্ডার ওয়ার্নিং দিলে More Info > Run Anyway)।
  3. ইনস্টল হয়ে গেলে ডেক্সটপে কীবোর্ড এর টপবার দেখতে পাবেন, মাইনাস আইকনে ক্লিক করলে নিচে টাস্কবারের ডানদিকে (System Tray) কীবোর্ডের আইকন দেখতে পাবেন।

⌨️ ৪. কীবোর্ড শর্টকাট

  • Ctrl + 1 : বাংলা ইউনিকোড
  • Ctrl + 2 : ইংরেজি
  • Ctrl + 3 : বিজয় আন্সি
  • Ctrl + 4 : বাংলা ফনেটিক
  • Ctrl + 5 : আরবি
  • Ctrl + Space : বাংলা ও ইংরেজির মাঝে সুইচ করতে।

💡 ৫. ফনেটিক টাইপিং গাইড ও মডিফায়ার

ফনেটিক টাইপিংয়ের ক্ষেত্রে যুক্তবর্ণ, কার এবং স্পেশাল ক্যারেক্টার লেখার কিছু স্মার্ট নিয়ম রয়েছে। নিচে সবগুলো শর্টকাট এবং ট্রিকস তুলে ধরা হলো:

১. যুক্তবর্ণ তৈরি ও ভাঙা (Zero Width Non-Joiner)

  • যুক্তবর্ণ তৈরি: পাশাপাশি দুটি ব্যঞ্জনবর্ণ লিখলে তা স্বয়ংক্রিয়ভাবে যুক্ত হয়ে যায়। যেমন: shikkha = শিক্ষা।
  • যুক্তবর্ণ ভাঙা: যদি দুটি ব্যঞ্জনের মাঝে যুক্তবর্ণ না চান, তবে তাদের মাঝে o (ও) চাপুন। এটি একটি ইনভিজিবল কার হিসেবে কাজ করে। যেমন: kko = ক্ক, কিন্তু koko = কক।
  • ZWNJ বা জয়েনার ভাঙা: আপনি চাইলে // (ডাবল স্ল্যাশ) অথবা ~ (Tilde) কী চেপেও যুক্তবর্ণ ভাঙতে পারেন। যেমন: kor//te = কর‌তে।
  • হসন্ত + ZWNJ: দুটি ব্যঞ্জনের মাঝে স্পেস না দিয়েও আলাদা দেখাতে চাইলে `` (ডাবল ব্যাকটিক) চাপুন। এটি 'হসন্ত + ZWNJ' বসাবে। যেমন: kos``meTiks = কস্‌মেটিকস্।
  • সাধারণ স্ল্যাশ: ৩ বার স্লাশ /// দিলে সেটি সাধারণ // হিসেবে প্রিন্ট হবে।

২. হসন্ত, রেফ এবং র-ফলা

  • হসন্ত (্): হসন্ত দিতে q অথবা ` (Backtick) অথবা *g ব্যবহার করুন।
  • র-ফলা (্র): ব্যঞ্জনবর্ণের পরে 'r' লিখলে র-ফলা হবে। যেমন: kro = ক্র।
  • রেফ (র্): ব্যঞ্জনবর্ণের আগে 'rr' লিখলে রেফ হবে। যেমন: rrk = র্ক।
  • ঋ-কার (ৃ): 'rri' লিখলে ঋ-কার হবে। যেমন: krri = কৃ।

৩. য-ফলা এবং ব-ফলা

  • য-ফলা (্য): ব্যঞ্জনবর্ণের পর z, y বা Z (Shift+Z) লিখলে য-ফলা হবে। যেমন: kz = ক্য। (Z সবসময় ফোর্সড য-ফলা হিসেবে কাজ করবে)।
  • ব-ফলা (্ব): ব্যঞ্জনবর্ণের পর w লিখলে ব-ফলা হবে। যেমন: sw = স্ব। (কিন্তু শুধু 'w' লিখলে 'ও' হবে)।

৪. স্মার্ট ম্যাক্রো ও স্পেশাল ক্যারেক্টার

  • ক্স / এক্স (x/X): শব্দের শুরুতে 'x' বা 'X' লিখলে হবে এক্স (যেমন: xray = এক্সরে)। কিন্তু শব্দের মাঝে বা শেষে লিখলে হবে ক্স (যেমন: box = বক্স)।
  • জ-ফলা আকার (্যা/অ্যা): এর জন্য ae ব্যবহার করুন। যেমন: gyan = জ্ঞান, byatha = ব্যাথা।
  • অ্যা / এ্যা: aya = অ্যা, eya = এ্যা।
  • র‌্যা: rya অথবা rza লিখলে র‌্যা হবে।
  • ওয়া / ওয়ে / ওয়্যা: wa = ওয়া, we = ওয়ে, wae = ওয়্যা।

৫. যতিচিহ্ন (Punctuation) ও অল্টারনেট শর্টকাট

  • দাঁড়ি এবং ফুলস্টপ: একটি ডট . দিলে বাংলা দাঁড়ি (।) আসবে। আর ইংরেজি ফুলস্টপ লিখতে চাইলে ডাবল ডট .. দিন।
  • বিসর্গ এবং কোলন: একটি কোলন : দিলে বিসর্গ (ঃ) আসবে। আর ইংরেজি কোলন লিখতে চাইলে ডাবল কোলন :: দিন।
  • টাকা (৳): ডলার সাইন $ চাপলে টাকার চিহ্ন (৳) আসবে।
  • খণ্ড-ত (ৎ): TH অথবা *t লিখুন।
  • অনুস্বার (ং): ng ।
  • বিসর্গ (ঃ): : অথবা hh অথবা *l ।
  • চন্দ্রবিন্দু (ঁ): ^ অথবা A (Capital A) অথবা *a ।

🗺️ ৬. লেআউট ম্যাপিং (Interactive Keyboard)

নিচের বাটনগুলোতে ক্লিক করে কীবোর্ডের শিফট (Shift) ক্যারেক্টারগুলো দেখে নিতে পারেন।

🎙️ ৭. ওয়াইফাই ভয়েস টাইপিং

পিসিতে মাইক্রোফোন ছাড়াই মোবাইল দিয়ে স্ক্যান করে বাংলা, ইংরেজি ও আরবিসহ যেকোনো ভাষায় কথা বললে পিসিতে টাইপ হবে!

  1. কীবোর্ডের ট্রে মেনু থেকে ওয়াইফাই ভয়েস টাইপিং চালু করুন।
  2. স্ক্রিনে আসা QR কোডটি মোবাইল দিয়ে স্ক্যান করুন।
  3. মোবাইলে ওপেন হওয়া পেজে মাইক (🎤) আইকনে চাপ দিয়ে কথা বলুন।

🧰 ৮. অন্যান্য টুলস

  • লোকাল ফাইল ট্রান্সফার: কোনো ক্যাবল বা ডেটা খরচ ছাড়াই পিসি থেকে মোবাইলে ফাইল আদান প্রদান করুন (ওয়াইফাই রাউটারের স্পিডে)।
  • লাইভ কনভার্টার: ইউনিকোড ↔ বিজয় আন্সি (SutonnyMJ) লাইভ কনভার্ট করুন।
  • কাস্টম লেআউট: সেটিংস থেকে নিজের ইচ্ছেমতো ৩টি কীবোর্ড লেআউট তৈরি করুন।

🧰 ৯. Bangla Toolbox ইকোসিস্টেম

Bangla Toolbox হলো বাংলা ভাষা নিয়ে কাজ করার জন্য তৈরি করা একটি কমপ্লিট ওয়েব প্ল্যাটফর্ম, যেখানে ৫০+ এর বেশি প্রফেশনাল টুলস রয়েছে। এটি Bangla Keyboard CS এর সাথে সরাসরি কানেক্টেড!
